# auto_thaumcraft_research

神秘时代自动研究辅助工具，原始配置适配 GTNH 2.8.0 beta4。

## 使用方法

1. 将 `auto_thaumcraft_research.zip` 放入 Minecraft 的 `resourcepacks` 文件夹并启用。
2. 双击 `start.bat` 启动控制面板。
3. 多屏环境先点击“选择屏幕”，选择 Minecraft 所在的屏幕。
4. 在神秘时代研究界面点击“扫描研究”，或按 `Ctrl + 8`。
5. 确认黄色连接方案后，点击“开始连线”，或按 `Ctrl + 5`。
6. 自动拖动期间随时按 `Esc` 紧急停止。急停后必须重新扫描。

控制面板不会抢占 Minecraft 的前台焦点，并会在截图和自动拖动期间自动隐藏，避免遮挡识别区域或接收鼠标事件。拖动控制面板标题可以移动窗口位置。

## 坐标设置

默认 `gc.txt` 适用于 2560×1440、Windows 150% 缩放。其他分辨率可以在控制面板中点击“坐标设置”，编辑后使用“预览绿框”检查位置。

`gc.txt` 每行格式：

```text
起始X,起始Y,横向间距,横向数量,纵向间距,纵向数量,方框大小
```

四行依次代表：

1. 中间研究点阵第一层
2. 中间研究点阵第二层
3. 左侧要素列表
4. 右侧要素列表

坐标以当前所选屏幕的左上角为 `(0, 0)`。程序会在所选屏幕上截图、显示覆盖层并换算自动拖动坐标，因此副屏位于主屏左侧或使用负坐标也可以工作。移动 Minecraft 窗口、调整 GUI 缩放或切换分辨率后，需要重新预览并校准绿框。

坐标设置窗口还可以调整单次拖动时长以及扫描、连线快捷键；`Esc` 固定用于紧急停止，不能被其他功能占用。

旧的 `tools.py` 仍可单独运行，用于显示 `gc.txt` 对应的绿色框，并会自动重新读取配置。
